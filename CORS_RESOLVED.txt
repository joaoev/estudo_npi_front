════════════════════════════════════════════════════════════════════

✅ CAUSA

Frontend (http://localhost:5173)
tentava acessar Backend (http://localhost:8080)
diretamente, mas o navegador bloqueou por CORS.

════════════════════════════════════════════════════════════════════

✅ SOLUÇÃO IMPLEMENTADA

Arquivo 1: vite.config.mts
────────────────────────
Adicionado proxy para redirecionar requisições /api:

server: {
  proxy: {
    '/api': {
      target: 'http://localhost:8080',
      changeOrigin: true,
      rewrite: path => path.replace(/^\/api/, ''),
    },
  },
}

Arquivo 2: src/services/api.ts
──────────────────────────────
Alterado baseURL:

const API_BASE_URL = '/api'  // Antes: 'http://localhost:8080'
