🐛 TROUBLESHOOTING RÁPIDO

Problema: Dados não carregam
  ✓ Verifique se backend está em http://localhost:8080
  ✓ Abra DevTools (F12) → Network → procure por /produtos

Problema: CORS Error
  ✓ Configure CORS no backend para aceitar http://localhost:5173

Problema: Formulário não salva
  ✓ Verifique console (F12 → Console)
  ✓ Valide os campos obrigatórios

💡 DICAS

   Edite: src/services/api.ts

   F12 → Network tab → filtre por requests
   F12 → Console → check logs de erro
